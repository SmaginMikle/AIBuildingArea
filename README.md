# AIBuildingArea
Определение площади застройки по спутниковым снимкам
Шаги решения задачи:
1. Model.ipynb - для датасета https://project.inria.fr/aerialimagelabeling/ создана модель, подобраны параметры с учетом имеющейся видеокарты и технических возможностей. Модель рассчитана и сохранена. Параметры модели удовлетворительные: Val IoU: 0.7515 | Val Acc: 0.9619
2. Predict.ipynb - по сохраненной модели получены предсказания для тестовой выборки датасета.
3. PredictAny.ipynb - по сохраненной модели получены предсказания для снимков произвольного размера.
4. DataScale (папка) - вручную подготовленный датасет с масштабами
5. ModelScale2.0.ipynb - методом проб и ошибок подобрана модель для определения масштаба снимка (первая неудачная версия ModelScale.ipynb). Получены предсказания для снимков произвольных размеров.
6. Models (папка) - рассчитанные модели: сегментации и регрессии.
