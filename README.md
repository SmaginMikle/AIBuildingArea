# AIBuildingArea
Определение площади застройки по спутниковым снимкам

Шаги решения задачи:
1. Model.ipynb - для датасета https://project.inria.fr/aerialimagelabeling/ (спутниковые снимки 5000*5000) создана модель (**семантическая сегментация, ResNet18**), подобраны параметры с учетом имеющейся видеокарты и технических возможностей. Модель рассчитана и сохранена. Параметры модели на валидационнной выборке: Val IoU: 0.7515 | Val Acc: 0.9619
2. Predict.ipynb - по сохраненной модели получены предсказания для тестовой выборки датасета.
3. PredictAny.ipynb - по сохраненной модели получены предсказания для снимков произвольного размера (все модели работают с квадратными снимками, реализовано преобразование к размеру модели).
4. DataScale (папка) - **вручную подготовленный датасет с масштабами картинок** (длина и ширина картинки в метрах). В дальнейшем дообогащен  снимками исходного датасета (пункт 1)
5. ModelScale.ipynb - первая версия модели регрессии по определению масштаба снимка (ResNet18). Метрики неудовлетворительные, модель сильно переобучается. Train Loss: 353.8576 | Val Loss: 52623.3666
6. ModelScale2.0.ipynb - методом проб и ошибок подобрана модель и ее параметры для определения масштаба снимка (**упрощенная самописная CNN**). Получены предсказания для снимков произвольных размеров. Train Loss: 0.002292 | Val Loss: 0.001730 | MAE total: 185.7 
7. Models (папка) - рассчитанные модели: семантическая сегментация для выделения зданий (house_segmentation_model.pth) и регрессии (CNN) для получения масштаба снимка (best_simple_cnn.pth).
